# 🎬 MovieZone BD - Project Architecture & Developer Guide

এই প্রজেক্টটি FastAPI (ব্যাকএন্ড এপিআই ও মিনি অ্যাপ সার্ভার) এবং Aiogram 3 (টেলিগ্রাম বট লজিক) এর সমন্বয়ে একটি মডুলার আর্কিটেকচারে তৈরি করা হয়েছে। যেকোনো নতুন ডেভেলপার বা এআই অ্যাসিস্ট্যান্টের কোড এডিটিংয়ের সুবিধার্থে নিচে গাইডলাইন দেওয়া হলো।

---

## 📂 ডিরেক্টরি ম্যাপ (Project Directory Map)

```text
my_project/
│
├── config.py             # গ্লোবাল কনফিগারেশন, ডেটাবেস কানেকশন ও গ্লোবাল মেমোরি ক্যাশ
├── helpers.py            # ইমেজ এডিটিং, অথেনটিকেশন ও FFmpeg ইউটিলিটি ফাংশনসমূহ
├── html_template.py      # মিনি অ্যাপের সম্পূর্ণ ফ্রন্টএন্ড (HTML, CSS ও JavaScript)
├── main.py               # এন্ট্রি পয়েন্ট (এখান থেকে সার্ভার ও বটের পোলিং একসাথে রান হয়)
│
├── bot/
│   ├── __init__.py       # প্যাকেজ ইনিশিয়ালাইজার (ফাঁকা থাকবে)
│   ├── handlers.py       # টেলিগ্রাম বটের সব কমান্ড, কিওয়ার্ড রিপ্লাই ও FSM স্টেট লজিক
│   └── workers.py        # ব্যাকগ্রাউন্ড কিউ (Queue) প্রসেস ও অটো-ডিলিট ওয়ার্কার লুপ
│
└── api/
    ├── __init__.py       # প্যাকেজ ইনিশিয়ালাইজার (ফাঁকা থাকবে)
    └── routes.py         # মিনি অ্যাপের সব ইউজার ও অ্যাডমিন এন্ডপয়েন্ট (FastAPI Routers)
